1. What is Python?
Python is a general-purpose interpreted, interactive, object-oriented, and high-level
programming language. It was created by Guido van Rossum in 1991. Python source code is also available under the GNU General Public License . Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance.

2. What are Code Environments? Types?
Code Environment is place where a programmer write, edit, compile and
execute the programs.

2) Full IDE's- Spyder, Pycharm, PyDev ,,Google Collab etc
